Medidas de Tendencia Central

Estadística básica con ejemplos, propiedades y gráficas

Media Aritmética

Simbología:
n = número total de datos
xi = dato i-ésimo
̅x (barra sobre la x) = media aritmética (promedio)

Fórmula general: X̄ = (x₁ + x₂ + ... + xₙ) / n

Definición: Es la suma de todos los valores dividida entre la cantidad de datos. Es la medida más común de tendencia central y representa el punto de equilibrio de los datos.

¿Cuándo usarla? Cuando todos los valores tienen la misma importancia o peso.

Propiedades importantes:

  • Es única para un conjunto de datos.
  • La suma de las desviaciones respecto a la media es siempre cero: ∑(xᵢ - X̄) = 0.
  • Se utiliza en muchos análisis estadísticos por su simplicidad y buena representatividad cuando no hay valores extremos.

Ventajas: Fácil de calcular, interpretar y comparar.

Limitaciones: Muy sensible a valores atípicos (outliers).

Ejemplo paso a paso:

  • Datos: 10, 15, 20, 25, 30
  • Suma: 10 + 15 + 20 + 25 + 30 = 100
  • Cantidad de datos: n = 5
  • Media: 100 / 5 = 20

Fuente: Levin & Rubin, "Estadística para administración y economía", capítulo 3.

Media Ponderada

Simbología:
xi = dato i-ésimo
wi = peso o frecuencia del dato i-ésimo
̅xp = media ponderada
n = número total de datos

Fórmula general: X̄ = (x₁·w₁ + x₂·w₂ + ... + xₙ·wₙ) / (w₁ + w₂ + ... + wₙ)

Definición: Promedio en el que cada valor tiene un peso asignado según su importancia relativa.

¿Cuándo usarla? Cuando algunos valores son más relevantes, como en calificaciones con diferente porcentaje o en inventarios con productos de distinto valor.

Propiedades importantes:

  • Si todos los pesos son iguales, la media ponderada coincide con la media aritmética.
  • Los pesos deben ser valores positivos y conocidos con precisión para que el resultado sea válido.

Ventajas: Representa mejor la realidad cuando los datos no tienen el mismo peso.

Limitaciones: Requiere conocer los pesos con precisión.

Ejemplo paso a paso:

  • Notas: 70 (peso 1), 80 (peso 2), 90 (peso 3)
  • Multiplicaciones: 70×1 = 70, 80×2 = 160, 90×3 = 270
  • Suma ponderada: 70 + 160 + 270 = 500
  • Suma de pesos: 1 + 2 + 3 = 6
  • Media: 500 / 6 = 83.33

Fuente: Mario F. Triola, "Estadística descriptiva", capítulo 4.

Media Geométrica

Simbología:
n = número de datos
xi = dato i-ésimo (todos > 0)
X̄g = media geométrica

Fórmula general: X̄g = n√(x₁ · x₂ · ... · xₙ)

Definición: Se calcula multiplicando todos los valores y extrayendo la raíz n-ésima, donde n es la cantidad de datos.

¿Cuándo usarla? En tasas de crecimiento, porcentajes o datos proporcionales, como interés compuesto, índices bursátiles o retornos financieros.

Propiedades importantes:

  • Siempre es menor o igual que la media aritmética (desigualdad de medias).
  • No se puede usar si alguno de los datos es cero o negativo.

Ventajas: Menos influenciada por valores extremos que la media aritmética.

Limitaciones: No se puede usar si algún valor es cero o negativo.

Ejemplo paso a paso:

  • Datos: 2, 8
  • Producto: 2 × 8 = 16
  • Raíz cuadrada (porque n = 2): √16 = 4

Fuente: Levin & Rubin, "Estadística para administración y economía", capítulo 3.

Media Armónica

Simbología:
n = número de datos
xi = dato i-ésimo (todos > 0)
H = media armónica

Fórmula general: X̄h = n / (1/x₁ + 1/x₂ + ... + 1/xₙ)

Definición: Es el inverso del promedio de los inversos de los datos.

¿Cuándo usarla? Ideal para promediar razones o tasas, como velocidad promedio, rendimiento por hora o eficiencia.

Propiedades importantes:

  • Da más peso a los valores pequeños, por lo que suaviza el efecto de valores grandes.
  • No puede aplicarse si hay ceros en los datos.

Ventajas: Útil cuando se trabaja con promedios de tasas.

Limitaciones: No puede aplicarse si hay ceros en los datos.

Ejemplo paso a paso:

  • Datos: 2, 4
  • Inversos: 1/2 = 0.5, 1/4 = 0.25
  • Suma de inversos: 0.5 + 0.25 = 0.75
  • Media armónica: 2 / 0.75 = 2.67

Fuente: Mario F. Triola, "Estadística descriptiva", capítulo 4.

Mediana

Simbología:
n = número de datos
x(i) = dato ordenado en posición i
Mediana = valor central del conjunto ordenado

Fórmulas:

  • Si n es impar: Mediana = x(n+1)/2
  • Si n es par: Mediana = (xn/2 + x(n/2)+1) / 2

Definición: Valor que se ubica en el centro de un conjunto ordenado de datos, dividiendo el conjunto en dos partes iguales.

¿Cuándo usarla? Cuando los datos presentan valores atípicos o están sesgados, pues la mediana no se afecta por valores extremos.

Propiedades importantes:

  • Si el número de datos es impar, la mediana es el valor central.
  • Si es par, es el promedio de los dos valores centrales.
  • Es una medida robusta y resistente a valores atípicos.

Ventajas: No se ve afectada por valores extremos o sesgos.

Limitaciones: Puede ser menos representativa si el conjunto tiene pocos datos o poca variabilidad.

Ejemplo paso a paso:

  • Datos: 3, 5, 7, 9, 11 (ordenados)
  • n = 5 (impar), mediana = tercer valor = 7

Fuente: Levin & Rubin, "Estadística para administración y economía", capítulo 3.

Moda

Definición: Valor o valores que aparecen con mayor frecuencia en un conjunto de datos.

Fórmula (datos no agrupados):
Moda = valor con mayor frecuencia

Fórmula (datos agrupados):
Moda = Lmo + [(fmo − f1) / (2fmo − f1 − f2)] × h

Simbología:

  • Lmo: Límite inferior del intervalo modal
  • fmo: Frecuencia del intervalo modal
  • f1: Frecuencia del intervalo anterior al modal
  • f2: Frecuencia del intervalo siguiente al modal
  • h: Amplitud del intervalo

¿Cuándo usarla? Cuando se quiere conocer el valor más común o frecuente en un conjunto de datos.

Propiedades importantes:

  • Puede no existir (si todos los valores aparecen la misma cantidad de veces).
  • Puede haber más de una moda (bimodal, multimodal).
  • Es la única medida de tendencia central que puede aplicarse a variables categóricas.

Ventajas: Fácil de identificar y útil para datos categóricos.

Limitaciones: Puede ser poco representativa o no existir.

Ejemplo paso a paso:

  • Datos: 2, 4, 4, 6, 7, 7, 7, 9
  • Moda: 7 (aparece 3 veces)

Fuente: Mario F. Triola, "Estadística descriptiva", capítulo 4.
